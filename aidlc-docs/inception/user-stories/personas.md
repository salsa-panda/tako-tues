- **2 名の役割分担型ペルソナ**（Clarification 1 = D）

## Persona 1: タロウ（コア生活侵食体験者）

> **キャッチコピー**: "火曜日になると、なぜかタコスのことを考えてしまう男。気がついたら、依存していた。"

### Profile
| 項目 | 値 |
|------|-----|
| **氏名（仮）** | 山田 タロウ |
| **年齢** | 33 歳 |
| **職業** | バックエンドエンジニア（Web サービス企業、フルリモート） |
| **居住地** | 東京都内 1LDK 一人暮らし |
| **平日の食生活** | 自炊 20%、Uber Eats 60%、コンビニ 20%。**料理は面倒くさくて避けたい派**。決められないことが多い |
| **料理スキル** | 中の下（パスタ茹でる・卵焼き作る程度。本格調理はほぼ未経験） |
| **テック親和性** | 高（PWA / Web Push に抵抗なし） |
| **タコス経験** | 月 2〜3 回、外食でしか食べたことがない。**家で作ったことはない** |
| **可処分所得** | 月数千円のサブスクなら気にせず払える |

### Goals（達成したいこと）
- **食生活の意思決定を外注したい**（毎日の「何食べる？」がストレス）
- **タコスをもっと食べたい**が、自分で発注を継続できない
- **料理は本心では避けたい**が、おもしろいガジェット／サービスは積極的に試したい
- **Taco Tuesday が自分の火曜日のデフォルトになってほしい** — 抵抗するより馴染むほうが楽、という幸せな降伏を受け入れていく

### Pain Points（痛み）
- 平日の昼夕、何を食べるか決めるのに 10 分使い、結局コンビニ
- 「健康的に食べねば」というプレッシャーに疲れている（**ヘルスケア逆張り**に共感する素地）
- **料理を始めると 30 分以上は確実に消える**ので、平日夜は手を出したくない
- **火曜 21:00 が近づくと、まだ何もしていないのに少し不安になる自分に気づいている** — タコスが頭を占領し始めている証拠

### Behaviors / Habits
- スマホは常に枕元、Web Push は許可派
- ダッシュボード型 UI が好き（Linear、Notion、GitHub プロフィールなど）
- 平日昼 12:00 と夕 18:00 にスマホをチェックする時間が一定
- 火曜夜の 21:00 はだいたいリモート勤務終わりの息抜きタイム

### Motivations to Use タコ中
- 「決めなくていい」という解放感
- シュールな体験そのものを楽しみたい（ネタとして友人に話したい）
- **頭の中でタコスの存在感が日々増していく "認知占有" 体験を、メタ視点で楽しんでいる** — 「あれ、また火曜日が来るのを気にしてる」という自己観察
- **"最高じゃないですか" という幸せな降伏の感覚** — 依存が苦痛でなく快感であることに気づいていく
- "**気がついたらタコスを作っている生活**" への変化を、本人の意思とは無関係に強制される面白さ

### How タコ中 Affects タロウ
- **金曜夜**: 強制発注決定。来週 8 人前のパストールキットが月曜届くと通告される（キャンセル不可）。もう火曜日のことを考え始めている
- **月曜夜 21:00**: 材料が玄関に届く（モック）→ リポジトリ同梱の手書きアミーゴ口調レシピが Push される（"アミーゴ、まずトルティーヤを温めるんだ"）。冷蔵庫を開けるたびにタコスの材料が目に入る
- **月曜〜火曜**: ダッシュボードに **24 時間カウントダウンタイマー**が表示される。残り時間が頭から離れない
- **火曜 21:00 の罰**: 24 時間以内に「作った」記録がないと、Tシャツ（黒地に TACO×3）が**勝手に発注**される
- **平日通知**: メキシコ風キャラの誘惑 Push でタコス摂取を促される。Push を見るたびにタコスを思う
- **未摂取時**: 段階的にエスカレートする通知に責められる（コントロール喪失の笑い）
- **3 ヶ月後**: 気がついたら「火曜 = タコスを作る日」が自分の生活リズムになっている。「最高じゃないですか」と感じている

### Quote（ペルソナの口癖）
> "気がついたら火曜日がタコスの日になってた。
> 誰も強制してないのに……いや、思いっきり強制されてるわ。
> ……まあ、別にいいか。**最高じゃないですか。**"

### Story 上の役割
- **コア生活侵食ループ全 13 ストーリーの主役**（US-T01〜T04, T06〜T10, T12〜T15）
- ストーリーボードの「タロウの侵食される 1 週間〜12 週間」タイムラインの中心人物
- 「Taco Tuesday の日本化」という文化移植の最初の受益者・体現者

## Persona 2: ミナミ（料理映像バイラル拡散役 / Taco Tuesday 文化伝道師）

> **キャッチコピー**: "Taco Tuesday を日本に広める、SNS 時代の文化伝道師。毎週火曜、彼女のフォロワーはタコスのことを考える。"

### Profile
| 項目 | 値 |
|------|-----|
| **氏名（仮）** | 佐藤 ミナミ |
| **年齢** | 26 歳 |
| **職業** | 都内 IT 企業マーケター、副業で SNS インフルエンサー（X / Instagram / TikTok 合計フォロワー約 12,000） |
| **居住地** | 東京都内、ルームシェア |
| **食生活** | 外食中心 + **SNS 映え自炊**月数回。新しいサービスを試したがる |
| **料理スキル** | 中（**動画映え**を意識した盛り付け・調理動画は得意） |
| **テック親和性** | 中〜高（消費者向け SaaS は積極利用、コードは書かない） |
| **タコス経験** | 月 1 回程度、メキシカン居酒屋好き |
| **SNS 投稿傾向** | "シュール"、"逆張り"、"バズる UI"、"30 秒レシピ動画" を好む。スクショ + 短文ツイート、**TikTok / Reels の縦動画**が主戦場 |

### Goals
- **新しいバイラルネタを発掘して投稿したい**（フォロワー獲得・案件獲得につながる）
- 投稿がバズった瞬間の達成感
- 「変なサービスを早めに見つけて発信した人」というブランディング
- **料理動画ジャンルでもオリジナルアングルを開拓したい**（「勝手に届いた材料で作るシリーズ」）
- **"日本に Taco Tuesday を広めた人" というポジションを確立したい** — 毎週火曜日に投稿することで、フォロワーの中に「火曜 = タコス」という認知を植え付けていく

### Pain Points
- 競合インフルエンサーが先にネタを掴んでしまう
- 紹介してもバイラルしない「普通すぎる」サービスが多い
- 紹介画像／スクショを撮りやすい UI が少ない
- 自分で食材を選ぶと SNS 投稿の **シリーズ化が難しい**（毎回違うネタになる）

### Behaviors / Habits
- スクショ → クロップ → 投稿の流れが秒で終わる
- スマホ縦持ち主体、Web Push 通知のスクショを SNS にそのまま貼ることがある
- **キッチンに三脚 + 縦撮りスマホスタンドが常設**

### Motivations to Use タコ中
- 「キャンセル不可」「押すと発注量増加」のシュールさを SNS で晒したい
- 「強制バリエーションウィーク・全部マンゴー味」の通知 UI をバズらせたい
- **「勝手に届いた材料 + 手書きアミーゴ口調レシピで作る 30 秒動画」**を**毎週 Taco Tuesday シリーズ化**できる
  - 「今週はパストールでした」「今週は強制マンゴーウィーク 4 日目です」のような週次更新コンテンツ
- **"日本初の Taco Tuesday インフルエンサー" というポジションを構築する手段**として、タコ中が最適なコンテンツエンジンになる
  - フォロワー 12,000 人が毎週火曜日にミナミの投稿を見て「Taco Tuesday」という言葉を覚える

### How タコ中 Affects ミナミ
- **金曜夜**: 強制発注通告のフルスクリーンモーダルをスクショ → "来週もタコス強制されました 🌮" として SNS 投稿
- **月曜夜**: 材料が届く → 開封動画 + 同梱レシピを TikTok に投稿（「アミーゴが今週のレシピ送ってきた。毎週 Taco Tuesday やってます 🇲🇽」）
- **火曜夜まで**: 24 時間カウントダウンを SNS で実況、フォロワーと「作る／作らない」をスレッドで遊ぶ
- **作るとき**: 30 秒料理動画を投稿（縦持ち、レシピを字幕でオーバーレイ）— **Taco Tuesday 動画シリーズ**として蓄積
- **作らなかったとき**: Tシャツ発注通知のスクショを「**罰受けました**」というキャプションで投稿
- **平日通知**: メキシコ風キャラの誘惑 Push やエスカレーション通知をスクショ → "ヤバい通知シリーズ" として連投
- **キャンセル試行**: ボタンを押して発注量が増える反応をスクショ → "押したら増えた" 投稿でバズ狙い
- **3 ヶ月後**: フォロワーがミナミのプロフィールを見て「この人毎週火曜日にタコス作ってる」と認識。Taco Tuesday 文化がフォロワーコミュニティの中に芽生え始める

### Quote
> "これ、絶対バズるやつ。
> しかも毎週ネタが届くから、TikTok のシリーズ化できる。
> **日本に Taco Tuesday 広めるの、私でしょ。**"

### Story 上の役割
- **SNS バイラル系 4 ストーリーの主役**（US-M01, M02, M04, M05。v1.4 で US-M03 削除済み）
- ストーリーボードの「ミナミの料理映像 → バイラル波及 → Taco Tuesday 文化移植」サイドタイムラインの中心人物
- **ハッカソン審査員へのアピール役**（テーマ「人をダメにするサービス」のバイラル性 + 文化移植の説得力を担保）

## なぜこの 2 名なのか（ハッカソン視点 v2.0）

- **タロウ**: 「**人をダメにするサービス**」テーマの**当事者性**を担保 — 「Taco Tuesday が自分の火曜日のデフォルトになってほしい」という幸せな降伏の体験者。3 ヶ月後に「最高じゃないですか」と感じるエンドステートが、**認知占有という "ダメになり方" の正体**を体現する

- **ミナミ**: テーマの**バイラル性 + 文化移植の説得力**を担保 — 「毎週火曜日に Taco Tuesday 動画を投稿する」という行為がフォロワーの中に Taco Tuesday 文化を植え付けていく。**ミナミの SNS 活動 = Taco Tuesday の日本伝播経路そのもの**

- 2 名で「**台所に追い込まれる側（認知占有体験者）× その様子をコンテンツ化して文化移植する側（伝道師）**」という構造を作り、コンセプト「**Taco Tuesday を日本に根付かせる**」に 2 軸の説得力を与える

- **v2.0 追記**: タロウがダメになっていく過程（認知占有）をミナミが可視化・拡散することで、「日本に Taco Tuesday が根付く」という文化移植プロセスが完成する。二人は共犯関係にある
