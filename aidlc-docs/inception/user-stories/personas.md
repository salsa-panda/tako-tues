## 設計方針

- **2 名の役割分担型ペルソナ**（Clarification 1 = D）
- **コア生活侵食体験者**（タロウ）と **料理映像バイラル拡散役**（ミナミ）
- 両者ともタコ中ユーザー、コアコンセプト「**タコスが生活の一部になる**」を体現
- 「届くのはタコス完成品ではなく材料」の前提により、両ペルソナとも**台所に立たされる**
- ハッカソン審査員は別途ペルソナ化しない（コア体験で勝負する）

## Persona 1: タロウ（コア生活侵食体験者）

> **キャッチコピー**: "料理から逃げてきたのに、なぜか今、トルティーヤを温めている男。"

### Profile
| 項目 | 値 |
|------|-----|
| **氏名（仮）** | 山田 タロウ |
| **年齢** | 33 歳 |
| **職業** | バックエンドエンジニア（Web サービス企業、フルリモート） |
| **居住地** | 東京都内 1LDK 一人暮らし |
| **平日の食生活** | 自炊 20%、Uber Eats 60%、コンビニ 20%。**料理は面倒くさくて避けたい派**。決められないことが多い |
| **料理スキル** | 中の下（パスタ茹でる・卵焼き作る程度。本格調理はほぼ未経験） |
| **テック親和性** | 高（PWA / Web Push / GPT に抵抗なし、ChatGPT Plus 加入済み） |
| **タコス経験** | 月 2〜3 回、外食でしか食べたことがない。**家で作ったことはない** |
| **可処分所得** | 月数千円のサブスクなら気にせず払える |

### Goals（達成したいこと）
- **食生活の意思決定を外注したい**（毎日の「何食べる？」がストレス）
- **タコスをもっと食べたい**が、自分で発注を継続できない
- **料理は本心では避けたい**が、おもしろいガジェット／サービスは積極的に試したい

### Pain Points（痛み）
- 平日の昼夕、何を食べるか決めるのに 10 分使い、結局コンビニ
- 「健康的に食べねば」というプレッシャーに疲れている（**ヘルスケア逆張り**に共感する素地）
- ChatGPT との会話が業務寄りに偏り、生活がおろそかになっている
- **料理を始めると 30 分以上は確実に消える**ので、平日夜は手を出したくない

### Behaviors / Habits
- スマホは常に枕元、Web Push は許可派
- ダッシュボード型 UI が好き（Linear、Notion、GitHub プロフィールなど）
- 平日昼 12:00 と夕 18:00 にスマホをチェックする時間が一定
- 火曜夜の 21:00 はだいたいリモート勤務終わりの息抜きタイム

### Motivations to Use タコ中
- 「決めなくていい」という解放感
- シュールな体験そのものを楽しみたい（ネタとして友人に話したい）
- 自分の依存ループを観察するメタ的な楽しさ
- "**料理しないで済む生活**" から "**気がついたらタコスを作っている生活**" への変化を、本人の意思とは無関係に強制される面白さ

### How タコ中 Affects タロウ
- **金曜夜**: 強制発注決定。来週 8 人前のパストールキットが月曜届くと通告される（キャンセル不可）
- **月曜夜 21:00**: 材料が玄関に届く（モック）→ Bedrock 生成のレシピが Push される（"アミーゴ、まずトルティーヤを温めるんだ"）
- **月曜〜火曜**: ダッシュボードに **24 時間カウントダウンタイマー**が表示される
- **火曜 21:00 の罰**: 24 時間以内に「作った」記録がないと、Tシャツ（黒地に TACO×3）が**勝手に発注**される
- **平日通知**: メキシコ風キャラの誘惑 Push でタコス摂取を促される
- **未摂取時**: 段階的にエスカレートする通知に責められる
- **ChatGPT GPT**: 業務会話の合間にタコ中 GPT を起動、生活の話題をタコス文脈に引き戻される
- **生活全体への侵食**: 冷蔵庫を開ければトルティーヤ、スマホを見れば誘惑 Push、ChatGPT を開けば説教、SNS にもタコ中の話題（後述ミナミ経由）

### Quote（ペルソナの口癖）
> "もう、何食べるか考えるの面倒なんだよ。勝手にタコスが来るならそれでいい。
> ……いや、待って、これ自分で作るやつ？"

### Story 上の役割
- **コア生活侵食ループ全 12 ストーリーの主役**（材料受け取り→レシピ配信→24時間カウントダウン→Tシャツ罰／サルサ通知／誘惑Push／GPT説教 等）
- ストーリーボードの「タロウの侵食される 1 週間」タイムラインの中心人物

## Persona 2: ミナミ（料理映像バイラル拡散役）

> **キャッチコピー**: "勝手に届いた材料で 30 秒のレシピ動画を作る、SNS 時代のメキシコ伝道師。"

### Profile
| 項目 | 値 |
|------|-----|
| **氏名（仮）** | 佐藤 ミナミ |
| **年齢** | 26 歳 |
| **職業** | 都内 IT 企業マーケター、副業で SNS インフルエンサー（X / Instagram / TikTok 合計フォロワー約 12,000） |
| **居住地** | 東京都内、ルームシェア |
| **食生活** | 外食中心 + **SNS 映え自炊**月数回。新しいサービスを試したがる |
| **料理スキル** | 中（**動画映え**を意識した盛り付け・調理動画は得意） |
| **テック親和性** | 中〜高（消費者向け SaaS は積極利用、コードは書かない） |
| **タコス経験** | 月 1 回程度、メキシカン居酒屋好き |
| **SNS 投稿傾向** | "シュール"、"逆張り"、"バズる UI"、"30 秒レシピ動画" を好む。スクショ + 短文ツイート、**TikTok / Reels の縦動画**が主戦場 |

### Goals
- **新しいバイラルネタを発掘して投稿したい**（フォロワー獲得・案件獲得につながる）
- 投稿がバズった瞬間の達成感
- 「変なサービスを早めに見つけて発信した人」というブランディング
- **料理動画ジャンルでもオリジナルアングルを開拓したい**（「勝手に届いた材料で作るシリーズ」）

### Pain Points
- 競合インフルエンサーが先にネタを掴んでしまう
- 紹介してもバイラルしない「普通すぎる」サービスが多い
- 紹介画像／スクショを撮りやすい UI が少ない
- 自分で食材を選ぶと SNS 投稿の **シリーズ化が難しい**（毎回違うネタになる）

### Behaviors / Habits
- スクショ → クロップ → 投稿の流れが秒で終わる
- スマホ縦持ち主体、Web Push 通知のスクショを SNS にそのまま貼ることがある
- ChatGPT GPT を発見したら共有リンクを即座に拡散
- **キッチンに三脚 + 縦撮りスマホスタンドが常設**

### Motivations to Use タコ中
- 「キャンセル不可」「押すと発注量増加」のシュールさを SNS で晒したい
- 「強制バリエーションウィーク・全部マンゴー味」の通知 UI をバズらせたい
- ChatGPT GPT の説教モード会話をスクショして共有したい
- **「勝手に届いた材料 + Bedrock 生成レシピで作る 30 秒動画」**を**シリーズ化**できる
  - 「今週はパストールでした」「今週は強制マンゴーウィーク 4 日目です」のような週次更新コンテンツ

### How タコ中 Affects ミナミ
- **金曜夜**: 強制発注通告のフルスクリーンモーダルをスクショ → SNS 投稿
- **月曜夜**: 材料が届く → 開封動画 + Bedrock レシピを TikTok に投稿（「アミーゴが今週のレシピ送ってきた」）
- **火曜夜まで**: 24 時間カウントダウンを SNS で実況、フォロワーと「作る／作らない」をスレッドで遊ぶ
- **作るとき**: 30 秒料理動画を投稿（縦持ち、Bedrock レシピを字幕でオーバーレイ）
- **作らなかったとき**: Tシャツ発注通知のスクショを「**罰受けました**」というキャプションで投稿
- **平日通知**: メキシコ風キャラの誘惑 Push やエスカレーション通知をスクショ → "ヤバい通知シリーズ" として連投
- **キャンセル試行**: ボタンを押して発注量が増える反応をスクショ → "押したら増えた" 投稿でバズ狙い
- **ChatGPT GPT**: 説教モードの会話をスクショ＋ GPT 共有リンクをセットで拡散

### Quote
> "これ、絶対バズるやつ。
> しかも毎週ネタが勝手に届くから、TikTok のシリーズ化できる。最高。"

### Story 上の役割
- **SNS バイラル系 4 ストーリーの主役**（強制発注スクショ拡散 / 材料開封＋レシピ動画投稿 / Tシャツ罰の "罰受けました" 投稿 / GPT 説教モード共有）
- ストーリーボードの「ミナミの料理映像 → バイラル波及 1 週間」サイドタイムラインの中心人物
- **ハッカソン審査員へのアピール役（テーマ「人をダメにするサービス」のバイラル性 + コンテンツ化可能性を証明）**

## なぜこの 2 名なのか（ハッカソン視点 v1.1）

- **タロウ**: 「**人をダメにするサービス**」テーマの**当事者性**を担保 — エンジニアの自己投影でリアリティのある依存ループを描ける。**料理から逃げたい像**にしたことで「材料が届いて作らされる」コンセプトの強制性が際立つ
- **ミナミ**: テーマの**バイラル性 + コンテンツ化可能性**を担保 — 「勝手に届いた材料で 30 秒レシピ動画」というコンテンツ軸が、SNS で**シリーズ化**できる構造を作る
- 2 名で「**台所に追い込まれる側 × その様子をコンテンツ化する側**」という構造を作り、コンセプト「**タコスが生活の一部になる**」に 2 軸の説得力を与える
- ペルソナの食生活・料理スキル設定により、コンセプト「**料理から逃げてきた人を台所に立たせる**」シュールさが直接ストーリーに反映される
